リスニング問題の音声指定方法

優先順位: audio_url（音声ファイル）→ audio_text（TTS読み上げ） 

{
  "type": "listening",
  "question": "音声を聞いて答えてください",

  // ① 音声ファイルURL（MP3/WAV/OGGなど）
  "audio_url": "https://example.com/audio/hello.mp3",

  // ② TTSフォールバック（URLが使えない場合）
  "audio_text": "Hello",
  "lang": "en-US",

  "options": ["Hello","Goodbye","Thanks","Sorry"],
  "answer": 0,
  "explanation": "解説（任意）"
}

audio_urlが設定されている場合はそちらを優先再生します。
失敗した場合は自動的にTTSにフォールバックします。
